03 / ОСНОВНОЙ ПРОЕКТ

WIZE

Корпоративная платформа, которая переводит бизнес-процесс из последовательности ручных действий в управляемый автоматический сценарий

Описать один раз - выполнять автоматически

После поступления заявки WIZE может создать задачу менеджеру, направить её на согласование, передать следующему отделу и уведомить клиента. Сотрудники сосредотачиваются на содержательной работе, а не на передаче информации между подразделениями.

Поскольку процессы компаний отличаются, платформа не пытается заранее реализовать все сценарии. Пользователи и партнёры могут подключать независимые модули со своей бизнес-логикой, backend и базой данных.

  • Multitenancy
  • Open API
  • Extensions
  • RBAC
  • Workflow

Пример автоматизированного сценария

  1. 01Поступление заявки
  2. 02Создание задачи менеджеру
  3. 03Согласование документов
  4. 04Передача следующему отделу
  5. 05Уведомление клиента

Платформенная модель: разработчики создают и продают модули, а каждый новый модуль повышает ценность WIZE для следующих компаний.
